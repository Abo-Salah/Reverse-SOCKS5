# Reverse-SOCKS5

Server/Client SOCKS5 (Secured Over Credential-based Kerberos 5) in Reverse mode on Windows.
Client need Proxifier for to work.
Defined in RFC 1928.

Contains:
- TCP connection
- Multi-Threads
- Kill-switch for server with a special request URL
